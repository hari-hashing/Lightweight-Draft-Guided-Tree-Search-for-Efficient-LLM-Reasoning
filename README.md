# Lightweight-Draft-Guided-Tree-Search-for-Efficient-LLM-Reasoning
Presents a new integrated inference technique which involves a combined approach of Speculative Decoding and MCTS 

# Install requirments using
First create a conda env using :
```python
conda create --name <environment_name> python>=3.12 

// activate the env

conda activate <environment_name>
```
or <u>create a python env</u> using :

```python
python=3.12 -m venv <environment_name>
```

Activate using

```bash
.\<environment_name>\bin\activate.bat
```

**OR** if created in linux,ubuntu or wsl use :

```bash
source <environment_name>\bin\activate 
```

THEN 

```python
pip install -r requirments.txt
```

# System Requirments 

Atleast **A40 GPU** with ~ 46GB VRAM (DRAM or HBM) or <u>HIGHER</u>
